# README

## Application Name: PHPウェブサイト


## Overview
PHPのみを使いCURD処理機能を実装したサイト作成
　

## [User]機能
```
1. 記事の閲覧・投稿・削除・編集
2. 会員登録
3. ログイン・ログアウト
4. お問い合わせフォーム
5. 記事の検索(タイトル＆本文）
```



## [Administrator]機能
```
1. 管理者は全ての機能を操作することができる
MySQLに接続しDB管理
```
